# Robot Sender Window UI

260630 AMD#1

独立桌面窗口应用原型，用于管理钉钉、飞书、企业微信等自定义机器人，并配置文本、附件、@ 人和定时/定期发送任务。

## 打开

桌面窗口：

```powershell
.\run-window.ps1
```

首次运行会自动安装 Electron 依赖。

浏览器预览仍可用：

```powershell
.\launch.ps1
```

## 当前实现

- 左侧机器人列表、平台分组、搜索、新增、复制配置。
- 机器人图标、名称、备注、Webhook 编辑与平台识别。
- 登录/权限参数表单与模拟权限检查。
- 发送消息、附件、@ 人、单次/定时/定期发送配置。
- 日期占位符预览，例如 `{today_yymmdd}`、`{yyyyMMdd}`。
- 配置导出为 JSON。
- Electron 独立窗口壳、自定义标题栏、窗口控制按钮。
- 桌面窗口中可用原生文件选择器取得真实本地路径。
