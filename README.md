# DingTalk-Robot-Manager

260701 AMD#2

Robot Manager 独立桌面窗口应用原型，用于管理钉钉、飞书、企业微信等自定义机器人，并配置文本、附件、@ 人和单次/定时发送任务。

## 打开

桌面窗口：

```powershell
.\run-window.ps1
```

首次运行会自动安装 Electron 依赖。

浏览器预览仍可用：

```powershell
.\launch.ps1
```

## 当前实现

- 左侧机器人列表、平台分组、搜索、新增。
- 左侧机器人右键菜单：机器人档案、重命名、修改图标、修改类型、备注、复制、删除。
- 机器人档案弹窗：图标、名称、类型、备注、Webhook 编辑与平台识别。
- 登录/权限参数表单与模拟权限检查。
- 权限检查卡片可点击查看需要的权限、参数和钉钉官方文档链接。
- 发送消息、附件、@ 人、单次/定时发送配置。
- 登录权限卡片会按不支持优先、同类按默认优先级排序。
- 配置导出为 JSON。
- Electron 独立窗口壳、自定义标题栏、窗口控制按钮。
- 桌面窗口中可用原生文件选择器取得真实本地路径，并可选择本地图标文件。
